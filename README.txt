pt_BR:
Para jogar o jogo, utilize as setas do seu teclado.
Aviso!: O jogo foi feito na versão python 3.13.12 com a bibliotéca pygame, está biblioteca até o presente momento não é compativel com python 3.14.
Para jogar é necessário instalar as bibliotécas listadas no arquivo "requirements.txt".
(Se for jogar pelo VS Code, não esqueça de mudar o interpretador para uma versão compativel com python 3.13.12)

en_US:
To play the game, use the arrow keys on your keyboard.
Warning!: The game was made in Python version 3.13.12 with the pygame library; this library is not currently compatible with Python 3.14.
To play, you need to install the libraries listed in the "requirements.txt" file.
(If you're playing using VS Code, don't forget to change the interpreter to a version compatible with Python 3.13.12)
